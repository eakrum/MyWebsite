WebRTC and Live Streaming



StevensLIVE

Mobile Application

Beta

My team of fellow classmates and I helped create a live streaming mobile app for students to never miss lecture. The application has the ability to track students in each lecture as well as having a built in chat feature. Thanks to Google's open source technology WebRTC, we were able to virtually reach real time live streaming video with a latency of 300 ms and a theoretical amount of 256 concurrent connections. The mobile application was built using React-Native, a framework to build android and iOS apps simulatenously, and our backend signaling server was built using Node.JS and hosted on an Amazon Web Services EC2 instance. Databases, storage buckets and authentication were built using FireBase which is a Google Cloud Platform product but the team is later looking to incorporate the AWS mobile hub instead to create a trusted service backed by the Amazon Web Services cloud.


StevenLIVE was a Verizon sponsored project and showcased at the annual Stevens Innovation Expo as a product created by the ECE Department.
